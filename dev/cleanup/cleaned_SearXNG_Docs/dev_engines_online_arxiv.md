# arXiv
arXiv is a free distribution service and an open-access archive for nearly 2.4 million scholarly articles in the fields of physics, mathematics, computer science, quantitative biology, quantitative finance, statistics, electrical engineering and systems science, and economics.
The engine uses the [arXiv API](https://info.arxiv.org/help/api/user-manual.html). 

**engines.arxiv.arxiv_search_prefix** = `'all'`
Search fields, for more details see, [Details of Query Construction](https://info.arxiv.org/help/api/user-manual.html#51-details-of-query-construction). 

**engines.arxiv.base_url** = `'https://export.arxiv.org/api/query'`
[arXiv API](https://info.arxiv.org/help/api/user-manual.html) URL, for more details see [Query-Interface](https://info.arxiv.org/help/api/user-manual.html#_query_interface)
